# Images Directory

Store all project images in this folder.

## Required Images

### Hero Section
- `hero-background.jpg` - Main hero background image (construction site, 2000x1200px recommended)

### About Section
- `team-photo.jpg` - Construction team photo (1200x1500px recommended, portrait orientation)

### Service Cards (800x600px recommended for each)
- `service-remodeling.jpg` - Kitchen/bathroom remodeling
- `service-flooring.jpg` - Flooring installation
- `service-drywall.jpg` - Drywall and painting
- `service-tile.jpg` - Custom tile work
- `service-framing.jpg` - Framing and carpentry
- `service-decks.jpg` - Decks and siding

## Image Guidelines
- Use high-quality, professional photos
- Optimize for web (compress images to reduce file size)
- Maintain consistent aspect ratios for similar image types
- Use descriptive file names
