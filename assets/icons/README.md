# Icons Directory

Store custom icon files (SVG format) in this folder.

## Custom Icons
Place any custom brand icons or specialized graphics here:
- `logo.svg` - Company logo
- `logo-white.svg` - White version of logo for dark backgrounds
- `favicon.ico` - Browser tab icon

## Note
The project currently uses Lucide React icon library for UI icons.
This folder is for custom/brand-specific icons only.

## SVG Guidelines
- Use SVG format for scalability
- Optimize SVGs (remove unnecessary metadata)
- Use consistent viewBox dimensions
- Keep file sizes small
