# Videos Directory

Store all project videos in this folder.

## Suggested Videos
- `hero-background.mp4` - Optional hero section background video
- `project-showcase.mp4` - Portfolio/project showcase videos
- `testimonial-clips.mp4` - Customer testimonial videos

## Video Guidelines
- Use web-optimized formats (MP4, WebM)
- Compress videos to reduce file size
- Include both video and fallback image options
- Recommended resolution: 1920x1080 (Full HD) or lower for faster loading
